# Student Management System

## 🚀 Features
- JWT Authentication (Login system)
- Full CRUD (Create, Read, Update, Delete)
- Search & Sorting
- Dashboard UI
- Session handling (auto logout)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Database: MySQL

## 📂 Project Structure
backend/ → API & DB logic  
frontend/ → UI
